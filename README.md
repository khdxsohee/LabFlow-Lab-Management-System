# LabFlow Diagnostics - Laboratory Management System

A modern, efficient, and secure Laboratory Management System (LMS) designed to streamline patient bookings, test catalog management, and diagnostic workflows.

## 🚀 Features

### 📊 Comprehensive Dashboard
- Real-time statistics for total bookings, pending reports, and delivered results.
- Daily and total revenue tracking.
- Visual overview of laboratory performance.

### 📝 Patient Booking Management
- Easy patient registration with unique IDs.
- Quick test selection from a pre-defined catalog.
- Status tracking (Pending/Delivered).
- Automated total amount calculation.

### 🧪 Test Catalog
- Manage your laboratory's test offerings.
- Categorize tests for better organization.
- Set and update pricing easily.

### 💬 WhatsApp Integration
- Send professional booking confirmations directly to patients via WhatsApp.
- Customizable message templates with dynamic placeholders (Patient Name, Booking ID, Amount).

### 🔐 Advanced Security
- **5-Digit Security PIN:** Protect your sensitive laboratory data with a custom numeric lock screen.
- **Local Persistence:** All data is stored securely in your browser's local storage.

### 💾 Data Management
- **JSON Backup:** Export your entire database (tests, bookings, settings) to a JSON file.
- **Data Restore:** Easily restore your data from a backup file.

### 🖨️ Professional Receipts
- Generate and print high-quality receipts for patients.
- Optimized for standard printing formats.

## 🛠️ Technologies Used

- **Frontend:** React 18+ with TypeScript
- **Styling:** Tailwind CSS (Utility-first CSS)
- **Icons:** Lucide React
- **Animations:** Framer Motion (motion/react)
- **State Management:** React Hooks (useState, useEffect, useRef)
- **Storage:** Browser LocalStorage API

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 📖 Usage Guide

1. **Setup Lab Details:** Go to the **Settings** tab to configure your Laboratory Name, Address, and Contact Number.
2. **Add Tests:** Navigate to **Test Catalog** to populate your available tests and pricing.
3. **Create Booking:** Use the **Dashboard** or **Bookings** tab to register a new patient and select their tests.
4. **Manage Security:** Set a 5-digit PIN in **Settings** to lock the application.
5. **Backup Regularly:** Use the **Backup Data** button in Settings to keep your data safe.
